JAVASCRIPT KOD YAPISI

JavaScript kodları HTML dili içerisinde kullanılır. <script> </script> etiketleri arasına JavaScript nesneleri eklenir.

<script>

alert("Merhaba Dünya!!");

<script/>

 

JavaScript kodları HTML belgesi içerisinde head ya da body elementleri arasında kullanılır.

Aşağıda head elementi arasında javaScript kodları kullanılmıştır.

<head>

<script>

Function ornek() {

document.getElementById(“ilkornek”).innerHTML = “Farklı metin!”;

</head>

 

JavaScript kodlarının bir diğer kullanım şekli ise direk olarak JavaScript dosyasını HTML belgesi üzerinden çağırarak kullanmaktır. Bu tür kullanımda dosyanız kod kalabalığından kurtulur.

<script src=”script.js”></script>

 

JavaScript kod yapısı hakkında bilmeniz gereken önemli diğer hususlar:

-JavaScript kodlarını kullanabilmek için HTML dili bilmek gerekir.

-BODY içerisinde kullanılan JavaScript kodları sayfa yükleme hızını artırır.

-Dışarıdan çağırılan JavaScript dosyası içerisinde <script> elementi kullanılmaz.
