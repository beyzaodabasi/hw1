For Döngüsü

Genellikle dizilerle çalışırken durum budur:

Yazmak yerine:

text += cars[0] + "<br>";

text += cars[0] + "<br>";

Yazabiliriz:

var i;

for (i = 0; i < cars.length; i++) {

text += cars[i] + "<br>";

}

 

for: Bir kod bloğundan birkaç kez geçer.

for/in: Bir nesnenin özellikleri arasında dolaşır.

for/of: Yinelenebilir bir nesnenin değerleri arasında dolaşır.

while: Belirtilen koşul geçerliyken bir kod bloğundan geçer.

do/while: Belirli bir koşul geçerliyken bir kod bloğundan da geçer.

 

For Döngüsü aşağıdaki sözdizimine sahiptir:

for (ifade 1; ifade 2; ifade 3) {

// Yürütülecek kod bloğu

}

 

İfade1, kod bloğunun yürütülmesinden önce (bir kez) yürütülür.

İfade1, kod bloğunu yürütme koşulunu tanımlar.

İfade1, kod bloğu yürütüldükten sonra (her seferinde) yürütülür.

 

for (i = 0; i < 5; i++) {

metin += "Sayı: " + i + "<br>";

}

 

While Döngüsü

While Döngüsü belirtilen bir koşul geçerli olduğu sürece bir kod bloğundan geçer.

while (koşul) {

// Yürütülecek kod bloğu

}

 

Aşağıdaki örnekte, bir değişken (i) 10'dan az olduğu sürece döngüdeki kod tekrar tekrar çalışacaktır:

while (i < 10) {

metin += "Sayı: " + i;

i++;

}

 

Do/While Döngüsü

Do/While Döngüsü döngü sırasında döngünün bir çeşididir. Bu döngü, koşulun doğru olup olmadığını kontrol etmeden önce kod bloğunu bir kez yürütür, ardından koşul doğru olduğu sürece döngüyü tekrarlar.

do {

// Yürütülecek kod bloğu

}

while (koşul);

 

Kod bloğu, koşul test edilmeden önce yürütüldüğünden, koşul yanlış olsa bile her zaman en az bir kez yürütülür:

do {

metin += "Sayı: " + i;

i++;

}

while (i<10);
